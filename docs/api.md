# Hello

Hello 是RT-Thread上的一个演示用package。

## API

### `int hello_func(void)`

hello 函数

| 参数              | 描述                                |
|:------------------|:------------------------------------|
|`无`               | 无参数                              |

| 返回值            | 描述                                |
|:------------------|:------------------------------------|
|0                  | 正确执行                            |
|-1                 | 失败                                |

#### 例子

``` c
#include <hello.h>

int func()
{
    hello_func();
    
    return 0;
}

```
